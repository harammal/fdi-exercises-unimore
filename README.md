# Fondamenti di Informatica Exercises

This repository contains my personal solutions to the exercises of the course "Fondamenti di Informatica" (Foundations of Computer Science) at the University of Modena and Reggio Emilia during the academic year 2024/2025.

These solutions are shared for educational purposes only.

## Repository Structure

- `01-basic-exercises/` – Introductory programming exercises
- `02-math-exercises/` – Mathematical problems
- `03-arrays/` – Array manipulation
- `04-command-line/` – Command-line based programs
- `05-strings/` – String processing
- `06-matrices/` – Matrix operations
- `07-file-handling/` – Reading and writing files
- `08-printing/` – Output formatting exercises
